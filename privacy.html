Privacy Policy

Bookmark Search is a Chrome extension designed to search your local browser bookmarks. This tool values your privacy and is built with a simple principle: your data stays with you.

What This Extension Does

This extension searches bookmarks already stored in your browser. It uses the chrome.bookmarks API to read bookmark titles and URLs, and displays search results in a popup interface.

What Data Is Collected

  • ✅ No data is collected
  • ❌ No analytics
  • ❌ No personal information
  • ❌ No external servers

Permissions

We request only the bookmarks permission in order to read and display your bookmark data locally. No other permissions are used.
