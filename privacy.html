Privacy Policy

Bookmark Search does not collect, store, or transmit any personal data or browsing activity.

What the extension does

The extension provides a local search interface for your existing browser bookmarks. All data access is done using the built-in chrome.bookmarks API, and all functionality is performed within your browser.

Data Handling

  • No user data is collected or stored
  • No data is sent to any external server
  • No cookies, trackers, or analytics are used

Permissions

The extension requests access only to your bookmarks in order to perform local search. No other permissions are requested.
